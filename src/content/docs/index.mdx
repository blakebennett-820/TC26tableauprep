## What You'll Build

In this hands-on training, you'll build a complete Tableau Prep flow that takes raw San Diego real estate data and transforms it into a predictive analytics dataset — all in one connected flow.

<CardGrid>
  <Card title="Connect & Cleanse" icon="pencil">
    Union files, remove nulls, fix spelling, and clean messy data using Tableau Prep and Prep Agent.
  </Card>
  <Card title="Pivot & Join" icon="random">
    Reshape data with pivots and combine multiple data sources with joins.
  </Card>
  <Card title="Predict with TabPy" icon="rocket">
    Run a Python-based predictive model directly inside your Prep flow using TabPy.
  </Card>
  <Card title="Spatial Analysis" icon="star">
    Create spatial points and join them to zip code geometries for geographic analysis.
  </Card>
</CardGrid>
