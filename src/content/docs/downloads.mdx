| File / Folder | Description |

| `starter_files_sheet1` (+ related files) | Raw real estate data for the Connect & Cleanse step |

| `predict_home_prices.py` | Python script for the TabPy predictive model |
| `ZIP_CODES.shp` | Spatial file for the Spatial Join step |
| `TC26_tabprep` | Pre-built Tableau Prep flow (for reference) |
| `TC26_workbook.twbx` | Completed Tableau workbook with final output |
